= Writing Mule Extensions

Extending Mule has never been so easy with *Mule DevKit*. There are two major Mule extensions types that can be built with the *Mule DevKit*:

[width="100%",cols="50%,50%",]
|===
|link:/anypoint-connector-devkit/v/3.3/writing-custom-modules[Modules] |Provide core capabilities to Mule applications such as transformation, filtering, routing, or custom business logic.
|link:/anypoint-connector-devkit/v/3.3/writing-custom-cloud-connectors[Cloud Connectors] |Connect Mule to SaaS services and on-premise application. Provides out of the box connection and session management.
|===
